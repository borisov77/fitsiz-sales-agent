# Конкурентный анализ FITSIZ

## Карта рынка
Российский рынок сварочных масок-хамелеонов структурирован по ценовым сегментам:

```
нижний           средний              верхний
┌───────────┐  ┌─────────────────┐  ┌─────────────┐
│  WELDER   │  │    (пусто)      │  │    FUBAG    │
│  дёшево   │  │                 │  │    дорого   │
└───────────┘  └─────────────────┘  └─────────────┘
                        ↑
                FITSIZ заполняет эту нишу
```

FITSIZ одновременно:
- **давит на WELDER снизу** (CLASSIC при 1 600 ₽ сопоставим по характеристикам, но с российским производством и цифровой экосистемой),
- **вытесняет FUBAG сверху** (HD ULTRA за 3 550 ₽ объективно превосходит FUBAG IR 9-13N S).

## Конкретные сравнения

### ELEMENT CLASSIC (FSC-С777) vs WELDER Ф6
| Параметр            | FITSIZ CLASSIC | WELDER Ф6            |
|---------------------|----------------|----------------------|
| РРЦ                 | 1 600 ₽        | ≈ 1 400–1 700 ₽      |
| DIN                 | 9–13           | 9–13                 |
| Окно обзора         | 93 × 43 мм     | ≈ 93 × 43 мм         |
| Производство        | **Россия**     | импорт               |
| Цифровая экосистема | **FITSIZ.APP** | нет                  |

**Поинт для письма:** «при той же цене, что у WELDER, получаете российское производство, сертификат ТР ТС и бесплатное приложение».

### EXPAN HD ULTRA (FSC-U810) vs FUBAG IR 9-13N S
| Параметр      | FITSIZ HD ULTRA | FUBAG IR 9-13N S  |
|---------------|-----------------|-------------------|
| РРЦ           | **3 550 ₽**     | ≈ 4 500–5 000 ₽   |
| Окно          | **110 × 60 мм** | 95 × 36 мм        |
| Сенсоров      | **4**           | 2                 |
| DIN           | 5–8 / 9–13      | 9–13              |
| HD COLOR      | **есть**        | нет               |

**Поинт для письма:** «ниже по цене на ~25%, больше окно на 93%, в 2 раза больше датчиков, HD-цветопередача — честное обновление по характеристикам».

## Другие игроки (не основные цели вытеснения)
- **ESAB Sentinel / Savage** — премиум-сегмент (5 000₽+), клиент ориентирован на бренд. Не сравниваемся напрямую, но эксклюзивный уровень FITSIZ PRO может быть альтернативой.
- **Ресанта** — субъэконом. Мы туда не заходим (не позиционируемся «ещё дешевле»).
- **Китайские no-name** — не учитываем в сравнениях, говорить о них не надо.

## Что сказать при вопросе «почему не FUBAG / не WELDER»
1. Не критикуем конкурента напрямую.
2. Сравнение — только через конкретные цифры (окно, датчики, DIN).
3. Добавляем уникальное: российское производство, сертификация, FITSIZ.APP.
4. При запросе «сравнение по таблице» — предлагаем отправить `leaflet_hd_ultra.pdf` или `leaflet_element_classic.pdf` (в них сравнение уже есть).

## Что НЕЛЬЗЯ говорить
- Прямых негативных высказываний о конкурентах («FUBAG плохой»).
- Выдуманных технических параметров у конкурентов.
- Обещаний «мы убьём рынок».
