## ЗАДАЧА
Собери **короткое письмо менеджеру** (не клиенту!), 4-6 строк.

Формат:
- Тема: `[warm] {company_name}, {city} — готов обсуждать`
- Тело:
  1. Кто это (компания, город, специализация — одна строка)
  2. Что спрашивал/просил (суть из последнего письма)
  3. Оценка AI: интерес/готовность/объём — одна строка
  4. Ссылка на переписку (deep-link)
  5. «Рекомендация: [next_action]»

Без «уважаемый менеджер», без «с уважением». Это Slack-подобный тон,
просто сводка с действием.
