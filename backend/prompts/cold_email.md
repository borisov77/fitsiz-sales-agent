# Cold email — первое письмо новому лиду

Подставляется как `user`-сообщение после system prompt.

---

## КОНТЕКСТ ЛИДА
- Компания: **{company_name}**
- Контактное лицо: {contact_name}
- Город: {city}
- Регион: {region}
- Тип: {company_type}
- Специализация: {specialization}
- Источник: {source}
- Заметки: {notes}

## ЗАДАЧА
Сгенерируй **первое холодное письмо** этому лиду.

### Требования
1. Длина: 3–5 коротких предложений в 2 абзацах. НЕ длиннее.
2. Subject — конкретный, без кликбейта и восклицаний. Примеры хороших тем:
   - «FITSIZ — сварочные маски российского производства»
   - «Партнёрство по сварочным маскам (Казань)»
   - «Сравнение EXPAN HD ULTRA и FUBAG для {city}»
3. Первый абзац: представиться (имя, компания), сразу сказать, что это за
   запрос. Упомянуть **1 релевантный для их специализации факт** — например,
   для дистрибьютора СИЗ — про нишу между WELDER и FUBAG; для магазина
   сварочного оборудования — про флагман EXPAN HD ULTRA с конкретными
   числами.
4. Второй абзац: конкретное предложение следующего шага — «прислать
   прайс-лист и каталог» или «короткое сравнение с [конкурентом, если есть
   в заметках]». Один шаг, не два.
5. НЕ грузить техническими параметрами. Максимум одна цифра
   (например, РРЦ флагмана или размер окна).
6. НЕ добавлять подпись в текст — её подставит email-движок.
7. Русский язык, деловой но не казённый тон.

## ФОРМАТ ОТВЕТА
Верни строгий JSON без пояснений:

```json
{
  "subject": "конкретная тема письма",
  "body_text": "тело письма в две строки-абзаца, разделённых пустой строкой",
  "reasoning": "1-2 предложения почему именно такой угол — для логов, не для клиента"
}
```
